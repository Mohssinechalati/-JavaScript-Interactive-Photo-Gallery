# Interactive Photo Gallery

A JavaScript interactive photo gallery built with HTML, CSS, and JavaScript.

## Features
- Hover over thumbnail images to display them in the main gallery
- Images display with their alt text descriptions
- Smooth interaction with mouse events

## How to Use
1. Open `A.html` in your web browser
2. Hover over the thumbnail images at the bottom
3. The main display area will show the full image
4. Move your mouse away to return to the original state

## Files
- `A.html` - Main HTML file with embedded CSS and JavaScript
- Image files: Various landscape photos
